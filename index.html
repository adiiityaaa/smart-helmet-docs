Smart Helmet for Rider Safety

Project Documentation

The Smart Helmet for Rider Safety is an embedded systems project developed to improve motorcycle rider safety by ensuring that the rider is in a suitable condition before and during the ride.

The system utilizes two Arduino UNO R3 boards communicating through Bluetooth, integrating alcohol detection, drowsiness detection, crash detection, and ignition control into a single safety system.

Project Highlights

Architecture

Dual Arduino UNO R3 System

Communication

HC-05 Bluetooth (Master-Slave)

Safety Features

Alcohol, Drowsiness & Crash Detection

Ignition Control

12V Relay Based Engine Lock
